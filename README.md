# Tratamento Psicológico Gratuito para Vítimas das Enchentes no Rio Grande do Sul 2024🤕

### Sobre o projeto
  Esse projeto foi desenvolvido durante a [`Imersão Dados_IA_2º_Edição`](https://www.alura.com.br/imersao-dados-ia), disponibilizada pela [`Alura`](https://www.alura.com.br/), mais especificamente, como desafio da aula 04. Nele, teve como objetivo, fornecer informações acessíveis e organizadas sobre opções de tratamento psicológico gratuito disponíveis para as pessoas que enfrentaram as enchentes no Rio Grande do Sul em 2024. A iniciativa visa auxiliar as vítimas a encontrarem o suporte necessário para lidar com o trauma, o estresse e outros desafios emocionais decorrentes dessa experiência traumática... Dentre algumas tecnologias apresentadas, estão: Gemini, Google AI Studio, Google Colab. Foi uma projeto que pontuou sobre o impacto significativo de como as inteligências artificiais podem auxiliar aquelas pessoas.
